# pinbot-hokamon
Pinning without permission 

---

## 概要  
メッセージ編集の権限がないメンバーでも、自由にピン留めを行えるようにするためのbotです。  
製作者の身内サーバのみでの使用を想定しています。  

---

## 使い方  
### メッセージをピン留めする  
ピン留めしたいメッセージに、📌(:pushpin:)のリアクションを付ける。

### メッセージのピンを外す  
ピン留めされたメッセージについている📌のリアクションを全て外す。  
または、3人以上のメンバーが 👎(:thumbsdown:)のリアクションを付ける。
